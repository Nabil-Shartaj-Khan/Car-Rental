For the frontend, in the terminal, type "cd car-rental" and then type "npm install" to install all the dependencies. After that, type "npm run dev" to start the frontend.

As for the backend, navigate to "car-rental-backend" and type "npm install" to install all the dependencies. Then, type "npm start" to run the backend server.

"Note: The backend server must run in order for the frontend to work."

Used: React.js (frontend), Express.js and Node.js(backend)
